# 模板文件命名规则

## 文件夹命名规则

所有语言文件夹使用**小写英文**命名，对应关系如下：

| 语言 | 文件夹名称 | 中文名称 |
|------|----------|---------|
| 英语 | `english` | 英语 |
| 阿拉伯语 | `arabic` | 阿拉伯语 |
| 土耳其语 | `turkish` | 土耳其语 |
| 西班牙语 | `spanish` | 西班牙语 |
| 斯洛伐克语 | `slovak` | 斯洛伐克语 |
| 越南语 | `vietnamese` | 越南语 |
| 法语 | `french` | 法语 |
| 亚美尼亚语 | `armenian` | 亚美尼亚语 |
| 印尼语 | `indonesian` | 印尼语 |
| 马来语 | `malay` | 马来语 |
| 波斯尼亚语 | `bosnian` | 波斯尼亚语 |

## 文件命名规则

每个语言文件夹内包含 **9个VIP等级** 的模板文件，命名格式为：

```
{Language}_vip{等级}.png
```

### 命名格式说明

- **{Language}**: 语言名称，使用**首字母大写**的英文名称
- **vip**: 固定小写，使用下划线连接
- **{等级}**: VIP等级，使用数字 1-9
- **文件扩展名**: `.png`

### 示例

#### 英语 (English)
```
templates/english/English_vip1.png
templates/english/English_vip2.png
templates/english/English_vip3.png
...
templates/english/English_vip9.png
```

#### 阿拉伯语 (Arabic)
```
templates/arabic/Arabic_vip1.png
templates/arabic/Arabic_vip2.png
templates/arabic/Arabic_vip3.png
...
templates/arabic/Arabic_vip9.png
```

#### 土耳其语 (Turkish)
```
templates/turkish/Turkish_vip1.png
templates/turkish/Turkish_vip2.png
templates/turkish/Turkish_vip3.png
...
templates/turkish/Turkish_vip9.png
```

## 完整文件列表

每个语言文件夹应包含以下9个文件：

1. `{Language}_vip1.png`
2. `{Language}_vip2.png`
3. `{Language}_vip3.png`
4. `{Language}_vip4.png`
5. `{Language}_vip5.png`
6. `{Language}_vip6.png`
7. `{Language}_vip7.png`
8. `{Language}_vip8.png`

## 注意事项

1. **文件夹名称必须小写**，例如：`arabic`、`turkish`
2. **文件名称首字母必须大写**，例如：`Arabic_vip1.png`、`Turkish_vip1.png`
3. **VIP等级使用数字**，从1到9
4. **使用下划线连接**，格式：`{Language}_vip{等级}.png`
5. **文件格式必须是PNG**
6. **所有文件必须放在对应的语言文件夹内**

## 文件路径示例

完整的文件路径格式：
```
templates/{语言文件夹}/{Language}_vip{等级}.png
```

例如：
- `templates/english/English_vip1.png`
- `templates/arabic/Arabic_vip1.png`
- `templates/turkish/Turkish_vip1.png`
